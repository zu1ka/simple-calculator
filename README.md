# Simple Calculator

A simple Node.js terminal calculator that performs addition, subtraction, multiplication, and division.  
This project uses [`prompt-sync`](https://www.npmjs.com/package/prompt-sync) for user input and handles invalid inputs.

## Features

- Add, subtract, multiply, and divide numbers
- Validates user input for the operation
- Defaults invalid or empty number inputs to `0`
- Prevents division by zero

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/zu1ka/simple-calculator.git

```
2. **Navigate into the project folder**
```bash
cd simple-calculator
```

3. **Install dependencies**
```bash
npm install
```
4. **Run the calculator using Node.js:**
```bash
node calculator.js
```
